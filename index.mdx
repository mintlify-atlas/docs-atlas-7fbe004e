Quiero Aprender ("I Want to Learn") is an interactive digital book created by and for children at [Platohedro](https://platohedro.org), a creative learning space in Medellín, Colombia. Built with Next.js 14, TypeScript, and React Three Fiber, it brings six hands-on experiences to life in the browser — from musical seeds and 3D character galleries to watercolor painting and cooperative board games.

The project was funded by the Presupuesto Participativo (Participatory Budget) of Comuna 9 – Buenos Aires, under the Narrativas Digitales y Expresión Cultural initiative.

## Six creative experiences

<CardGroup cols={3}>
  <Card title="Sound Seeds" icon="music" href="/experiences/semillas-sonoras">
    Tap seeds to hear unique audio textures. Powered by the Web Audio API with real-time volume control.
  </Card>
  <Card title="Marimba" icon="piano-keyboard" href="/experiences/marimba">
    Play a digital marimba with keyboard and click interaction, producing musical tones in the browser.
  </Card>
  <Card title="Character Gallery" icon="users" href="/experiences/galeria-personajes">
    A 3D gallery of fantastical characters created by children, rendered with React Three Fiber and FBX models.
  </Card>
  <Card title="Watercolor Painting" icon="paintbrush" href="/experiences/acuarela">
    Paint with Canvas 2D — colors blend and spread like real watercolor. Export your work as a PNG.
  </Card>
  <Card title="Lava Lamp Lab" icon="flask" href="/experiences/laboratorio-lava">
    A simulation of lava lamp physics using custom WebGL shaders. Observe, hypothesize, and wonder.
  </Card>
  <Card title="Self-Care Game" icon="dice" href="/experiences/juego-autocuidado">
    A cooperative 45-square board game with dice animations and self-care challenge cards.
  </Card>
</CardGroup>

## Getting started

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/platohedro/Quiero-aprender.git
    cd Quiero-aprender
    ```
  </Step>
  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>
  <Step title="Run the development server">
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) in your browser.
  </Step>
  <Step title="Explore the experiences">
    Navigate between the six chapters using the **Capítulos** dropdown in the top navigation bar.
  </Step>
</Steps>

<Note>
  The project is a living prototype. New experiences, content, and features are added as the Matinée community discovers new ideas. Contributions and adaptations are welcome — see the [Contributing](/contributing/setup) section.
</Note>
