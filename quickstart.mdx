This guide gets the full application running on your machine. You will need about five minutes and a working Node.js installation.

## Prerequisites

- **Node.js 18 or later** — check with `node -v`. Download from [nodejs.org](https://nodejs.org) if needed.
- **npm** (bundled with Node.js), or an equivalent package manager such as yarn or pnpm.
- **Git** to clone the repository.

<Note>
  The application interface is in Spanish. Navigation labels, chapter titles, button text, and game copy all appear in Spanish — this is intentional, as the project was created by and for children in Medellín, Colombia.
</Note>

## Setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/platohedro/Quiero-aprender.git
    cd Quiero-aprender
    ```
  </Step>
  <Step title="Install dependencies">
    Install the project's dependencies. The app uses Next.js 14, React Three Fiber, and three.js, so the initial download may take a moment.

    <CodeGroup>
      ```bash npm
      npm install
      ```
      ```bash yarn
      yarn install
      ```
      ```bash pnpm
      pnpm install
      ```
    </CodeGroup>
  </Step>
  <Step title="Start the development server">
    ```bash
    npm run dev
    ```

    Next.js compiles the app and starts a local server. You will see output like:

    ```
    ▲ Next.js 14.2.5
    - Local: http://localhost:3000
    ```
  </Step>
  <Step title="Open the app in your browser">
    Navigate to [http://localhost:3000](http://localhost:3000). The **Bienvenid@s** (Welcome) section loads first, showing a card grid with all six chapter experiences.
  </Step>
  <Step title="Navigate between experiences">
    Use the **Capítulos** dropdown in the top navigation bar to jump directly to any of the six chapters. You can also scroll through the page — the active chapter updates automatically as each section enters the viewport.

    Each chapter also has an **Abrir solo** (Open alone) button that opens the experience as a standalone page at `/sesiones/[id]`.
  </Step>
</Steps>

## Available scripts

The `package.json` defines three scripts:

```json
{
  "scripts": {
    "dev": "next dev",
    "build": "next build",
    "start": "next start"
  }
}
```

| Script | Command | What it does |
|---|---|---|
| Development | `npm run dev` | Starts Next.js in development mode with hot reload |
| Build | `npm run build` | Compiles a production-optimised bundle |
| Production | `npm run start` | Serves the compiled build (run `build` first) |

<Tip>
  Use `npm run dev` during local development. The Three.js 3D scenes and Web Audio API features work best in a Chromium-based browser (Chrome, Edge, Arc).
</Tip>

## What you'll see

The application is a single scrollable page divided into sections. The top navigation bar has two elements:

- **Inicio** — scrolls back to the welcome section with the six chapter cards.
- **Capítulos** — a dropdown listing all six chapters by name. Selecting one smooth-scrolls to that section and updates the URL hash.

The six chapter sections appear in this order:

| # | Section ID | Title (Spanish) | Experience |
|---|---|---|---|
| 01 | `semillas` | El cuidado de la semilla | Sound seeds and marimba |
| 02 | `galeria` | Creación de personajes fantásticos | 3D character gallery |
| 03 | `laboratorio` | Experimentos de ciencia con monstruos de lava | Lava lamp simulation |
| 04 | `mesa` | Juego de mesa sobre el autocuidado | Cooperative board game |
| 05 | `acuarela` | Pintura en acuarela | Watercolor canvas |
| 06 | `slot` | El animalario | Animal slot machine |

Below the chapters, the page includes **Sobre** (About) and **Créditos** (Credits) sections, followed by a footer with logos for Platohedro, Terre des Hommes, and the Alcaldía de Medellín.
